## Why This Repository Qualifies

I am the primary maintainer of this open source repository. The project focuses on reading Ga-In-Sn ternary phase diagram compositions with a Python desktop interface. It allows users to calibrate the Ga/In/Sn triangle vertices from a paper, chart, or experiment image, then calculates the clicked point composition using barycentric coordinates.

The repository is currently in an early open source stage, so stars and forks are still limited. Its value is in a clear scientific and educational use case: helping materials science students and alloy researchers estimate ternary composition values more quickly and with less manual reading error. I plan to keep improving documentation, examples, tests, packaging, and issue handling.
